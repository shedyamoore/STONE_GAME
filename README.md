this is a game to demonstrate the winner of a stone game

Consider a game board consisting of 4x4 cells,
        each of which contains a piece of stone.
        Players move in turn. In a move, a player must choose a (non empty) row or column and take away 1, 2, 3, or 4 stones from it.
        If 2 or more stones are removed, they must be adjacent. The winner of the game is the player who makes the last stone
